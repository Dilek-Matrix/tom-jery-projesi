# Tom and Jery Arayüz Çalışması

Bu proje, frontend geliştirme becerilerimi artırmak amacıyla yapılan bir **klon (çalışma)** projesidir.

### ℹ️ Bilgilendirme
Bu çalışma tamamen **eğitim amaçlı** hazırlanmıştır. Tasarım, logolar ve marka isimleri ilgili firmaya aittir. Herhangi bir ticari amaç gütmemektedir.

### Kullanılan Teknolojiler
* HTML5
* CSS3 (Grid/Flexbox)
